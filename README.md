基于PPGo_ApiAdmin改过来的基础后台管理框架

有什么价值？
----
1、RBAC权限完善，多角色管理系统    
2、后台界面完整，多标签页面
所以，可以作为一个基础框架使用，快速开发。初学者还可以作为熟悉beego使用。

用到了哪些？
----
1、界面框架layUI2.4.5    
2、makedown.md   
3、beego1.8
4、Ztree
